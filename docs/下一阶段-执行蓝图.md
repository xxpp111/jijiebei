## GPT-image-2 调用 how-to（照搬 CM 实践，可直接跑）
- 走字节 ModelHub 直连（**不走 Dubhe**，CM 文档明确 GPT-image-2 是例外）。
- endpoint：`https://aidp.bytedance.net/api/modelhub/online/v2/crawl/openai/images/generations`
- model：`gpt-image-2`；key：`export MODELHUB_AK=xxx`（URL `?ak=` 和 header `api-key` 都带）。
- 参数：`{model, prompt, n:1, size:"1536x1024", quality:"high"}`；返回 `data[0].b64_json` → base64 解码即 PNG。
- **打法**：让模型出**无字底图**（prompt 顶部大写 `CRITICAL — NO TEXT`），"集结杯"等中文用 Cocos Label 叠（避免中文乱码）。prompt 用结构化英文（CRITICAL 硬约束 + 分区 + 风格分点），参考 `CM/scripts/gen_announcement.py` 的 PROMPT。
- 最小封装（urllib，无三方依赖）：
```python
import base64, json, os, time, urllib.request
API="https://aidp.bytedance.net/api/modelhub/online/v2/crawl/openai/images/generations"
def gen_image(prompt, out, size="1536x1024", quality="high"):
    ak=os.environ["MODELHUB_AK"]
    body=json.dumps({"model":"gpt-image-2","prompt":prompt,"n":1,"size":size,"quality":quality}).encode()
    req=urllib.request.Request(f"{API}?ak={ak}", data=body,
        headers={"Content-Type":"application/json","api-key":ak,"X-TT-LOGID":f"jjb-{int(time.time())}"}, method="POST")
    p=json.loads(urllib.request.urlopen(req, timeout=600).read())
    open(out,"wb").write(base64.b64decode(p["data"][0]["b64_json"]))
```
- 建议背景图 prompt 要点：16:9（或按界面比例）、电竞赛事背景、深藏青+金属金/银、星际2 科幻、**NO TEXT 留标题位**、官方指挥官/残酷氛围。

## Claude Design 使用教程（claude.ai/design）
**边界**：它产出 web(React/HTML/CSS) 视觉稿/可点原型——我们用它**定视觉方向/布局/配色/出 mock**，不能直接进 Cocos；定稿后由 Claude Code 翻译成 Cocos 节点。正好用它探索 CM×集结杯 联名视觉，拿去和 CM 讨论。

**5 步**：
1. 打开 claude.ai/design 新建 design。
2. **喂上下文（决定质量）**：拖入参考图（当前 4 屏 UI 截图 + CM logo 图 + `CM 公告.tga`，拖拽或 ⌘V）；粘贴下方 prompt；可选 Import 挂载 `jijiebei` 文件夹让它看资产。
3. **生成+迭代**：先出"几版方向/线框"→选一版用 Prototype 推高保真→Comment 点元素批注（可批量）、Knobs 拖着实时调色/间距/字号。
4. **多出几版配色**拿去和 CM/土豆讨论选定。
5. **定稿后** Share 导出 或 "Handoff to Claude Code" 导出规范包 → 发我，我在 Cocos 落地。

**省力技巧**：≤3 色（navy+金+银，别再加）；⌘G 语音；Comment 批注批量发；Knobs 实时调；Handoff 给我落地。忽略贴士里的生活妙招。
