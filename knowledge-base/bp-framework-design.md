## 一、三层框架总览

```
[池层] 因子母池 = 全量官方因子(因子配置.txt) + 自定义因子(加行) − 排除名单 − 排除组合
       指挥官母池 = 指挥官配置.txt(A弱/B强 + 权重列) − ban名单(每周一A一B)
                              │
[金框层] 给因子打"金框=强化"标记（GOLD_FACTORS / 点金）
                              │
[BP层] 开局交互：多揉几个因子→选手 ban 掉不想要的 / 自选指挥官 / 二选一 or 都给（可配）
                              │
                        最终每场 因子组合 + 指挥官
```

赛制变 = 改配置，不动抽取内核。Image#20 两套方案、每周 ban、自选、ban 因子 都落进这框架当配置项。
